# GN Freight WebApp

**Stack:** Streamlit (Python), NumPy, Pandas

**Huvudfunktioner:**
- Calculator (kg/pall/FLM, auto-lowest)
- Kurvfit (POWER/EXP) + R²
- Tillägg: fuel, MARPOL, extra, per LDM/kg, flat + FR roadfee-hjälp
- LDM Scaler (1–13)
- Weight Scaler (retiering)
- Containers (1–19 från FTL=20)
